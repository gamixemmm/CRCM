# AtlasDrive Website

Standalone static marketing site for AtlasDrive, localized for Morocco.

## Files

- `index.html` - main page
- `styles.css` - all styling
- `script.js` - mobile navigation, sticky header, and number-input wheel guard
- `assets/atlasdrive-product.png` - hero/product visual

## Languages

The site supports French, Arabic, and English. French is the default language. Arabic switches the page to RTL automatically.

## Hosting

This folder does not depend on the dashboard app or Next.js server. You can move the `website` folder anywhere and host it as static files.

Open `index.html` directly in a browser or deploy the folder to any static host.
